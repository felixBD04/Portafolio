Portafolio personal **one-page** con estética **brutalist + dark mode neón**, animaciones de alto impacto y experiencia bilingüe (ES/EN). Construido con HTML, CSS y JavaScript vanilla — sin frameworks pesados — para que sea rápido, fácil de mantener y desplegable directamente en GitHub Pages.

## ✨ Características

- 🎨 **Diseño brutalist + neón** sobre azul profundo (`#0A0E1A`) con acentos cyan (`#00F0FF`) y magenta (`#B026FF`)
- 📱 **100% responsive** — mobile-first, breakpoints en 400px, 600px y 900px
- ⚡ **Animaciones de alto impacto** con [GSAP](https://greensock.com/gsap/) + ScrollTrigger
- 🌐 **Bilingüe ES/EN** con toggle en vivo (sin recargar)
- 🖱️ **Cursor personalizado** con lag suave (solo desktop)
- 🌧️ **Canvas Matrix rain** de fondo, optimizado y respetuoso con `prefers-reduced-motion`
- 🔤 **Text scramble** en el hero (las letras se descodifican)
- 🎮 **Easter eggs:** Konami code, escribir `sudo` (mini-terminal), 5 clicks en el nombre
- ♿ **Accesible:** skip-link, aria-labels, semántica correcta, contraste AA
- 🚀 **Performance:** SVG en lugar de PNG, lazy loading, fonts con preconnect
